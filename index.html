📦 CSS Container Queries

The Future of Responsive Design: Media queries respond to the viewport, but container queries respond to their parent component. This means components can be truly reusable and adapt to wherever they're placed.

Perfect for modern component-based architectures where widgets live in different layouts.

<!-- Container setup -->
<div class="card-container">
  <article class="card">
    This adapts to the container size!
  </article>
</div>

/* CSS Magic */
.card-container {
  container-type: inline-size;
}

.card {
  display: grid;
  grid-template-columns: 1fr; /* Stack vertically by default */
}

@container (min-width: 300px) {
  .card {
    grid-template-columns: repeat(2, 1fr); /* Side-by-side when space allows */
  }
}

Browsers supported this since ~2021. Stop fighting with percentage widths and embrace contextual responsive design!
